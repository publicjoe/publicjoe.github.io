In this part of the tutorial, we will do a little update to the style colours for dark mode

Dark Mode Consistency

I have tried to make the editors as vs-dark, but after a little research we need to make a small change. The Tabs and Gutter are light gray (#f5f5f5 and #ddd). So this change is aimed at unifying the theme.

style.css 
#tabs {
  background: #252526; /* Match Monaco dark background */
  border-bottom: 1px solid #3c3c3c;
}
#tabs button {
  color: #969696;
}
#tabs button.active {
  background: #1e1e1e;
  color: #fff;
  border-bottom: 2px solid #007acc;
}
.gutter {
  background-color: #333; /* Darker gutter */
}
