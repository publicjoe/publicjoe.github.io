Variables

Overview

Variables are placeholders or labels for a piece of information stored in the computer‚Äôs short term memory.

You give a variable a name and assign to it any information you might be using in your code: numbers, text, or more complex data structures like objects or arrays.

As the name suggests, variables can be changed and updated as our code executes.

const vs let

There are two primary keywords we use to declare a variable in JavaScript: const and let.

They‚Äôre similar, except const values are not allowed to change during the execution of the code.

You should probably use const to declare your variables, unless you‚Äôre absolutely sure that the value needs to change!
