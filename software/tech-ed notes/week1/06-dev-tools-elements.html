Dev Tools: Elements Tab and Computed Styles

Overview

Developer Tools in modern browsers are extremely effective at giving actionable insight into the current state of a web page, the data, events and animations occurring on the page.

The first tab we’ll look at is the “Elements tab” which is where you land when right clicking and selecting “Inspect Element” on a page. It shows the HTML document structure, plus the CSS styles applied to each element when selecting it.

The second tab we’ll look at is the “Computed Styles tab” which shows the final CSS rules applied to an element, including any rules inherited from parent elements.
