Discussion

What is the web?

The web is a network of computers, connected together, that can communicate with each other.

What is a web browser?

A web browser is a program that can make requests to a web server, and display the response to the user. Chrome, Firefox, Safari, etc.

What is a web server?

A web server is a program that can receive requests from a web browser, and send a response back to the browser.

What is HTML?

HTML is a markup language that allows us to describe the structure of a web page.

What is CSS?

CSS is a language that allows us to describe the appearance of a web page.

What is JavaScript?

JavaScript is a programming language that allows us to add interactivity to a web page.
