Events

Overview

Events are how we respond to user input and interaction on our page. If a user clicks a button or types into an input field, events are announced that we can listen for and respond to. There are many types of events from clicks to scrolling, key presses and more.

Class Plan

  • Demo: Demonstration of click events and how they work
  • Workshop: Write some event handling code

Learning Objectives

  • What is an event?
  • How to listen for events
  • How to handle events
  • How to respond to events

Success Criteria

  • I know what an event is
  • I can write JavaScript code to listen to events on elements on the page, which performs a task

Every element once selected has an addEventListener method that allows us to listen for events on the element and specify a block of code to run when it occurs.

‚õ≥Ô∏è Add an event listener to the button using the addEventListener method. Log to the console when the event occurs.

  console.log(event);
