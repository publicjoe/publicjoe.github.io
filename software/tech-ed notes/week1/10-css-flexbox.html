CSS: Flexbox

Overview

Flexbox is a layout system that allows you to arrange elements in a row or column. It is a powerful tool to help with creating responsive layouts.

Page template layout techniques on the web have come a long way in recent years with huge leaps in browser support for modern CSS features like Flexbox and Grid.

No one is building layouts with older layout approaches like floats, positioning, or HTML tables any longer (thankfully!). You might encounter some of them on old projects though!

Class Plan

    • Flex container/parent and direction, main axis, wrapping, justify content, alignment

  • Workshop: Create a website layout using Flexbox

Learning Objectives

  • What is Flexbox?
  • How to use Flexbox to create a page layout
  • Flexbox properties for the parent (display, axis direction, wrapping, justify and align content) and children (order, grow/shrink, align-self, auto margin)

Success Criteria

  • I know what display: flex; does to its immediate children
  • I can write CSS to move items along rows and columns
  • I know how to change the direction of a flex container
