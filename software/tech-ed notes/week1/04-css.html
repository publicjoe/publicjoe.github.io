CSS: Introduction to CSS

Overview

CSS, or Cascading Style Sheets, is a language of properties and values that we use in many combinations to achieve the look and feel of a website.

Every website on the internet uses some combination of CSS rules to produce the appearance of the site.

CSS is a powerful tool, simple to start with but extremely capable and useful in endless unexpected ways. The things people can achieve with CSS alone is often surprising!

Workshop

Time for you to start writing some CSS rules!

Remember that we write our HTML tags, elements and content in our index.html file.

We write our CSS selectors, classes and rules in a style.css file.

We link the two together in the <head> of our index.html with a <link> tag:

<head>
  <meta charset="UTF-8" />
  <meta name="viewport" content="width=device-width, initial-scale=1.0" />

  <!-- In the link tag below, the rel attribute defines the relationship between this file and the one we're linking to. -->
  <!-- the href attribute points towards my .css file -->
  <link rel="stylesheet" href="style.css" />

  <title>CSS Intro</title>
</head>
