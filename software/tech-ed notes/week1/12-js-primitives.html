Data types

Overview

Data is information that we store in the computer‚Äôs memory. It can be a number, some words, or something more complex.

We treat different types of data in different ways, because a number and a word are not the same thing.

If you have two numbers, 5 and 10, you can add them together using arithmetic and get 15 in return.

If you have two words, ‚Äúhello‚Äù and ‚Äúworld‚Äù, you can‚Äôt add them together in the same way. You can combine them together to make a new ‚Äústring‚Äù of words: ‚Äúhello world‚Äù.

If you‚Äôre dealing with something that‚Äôs true or false, you can‚Äôt add it to anything. True is true. You can‚Äôt add true to true and get more true.

The primitive data types in JavaScript are numbers, strings (text), and booleans (true or false).

The typeof Operator

You can use the typeof operator to find out what type of data a variable contains. You might use this to check if you have the correct type of information. A number you can add, or a string you can concatenate.

let name = "Bob";
console.log(typeof name); // Output: string

let number = 1;
console.log("The type of number is " + typeof number); // Output: The type of number is number

let isTrue = true;
console.log(`The type of isTrue is ${typeof isTrue}`); // Output: The type of isTrue is boolean

undefined and null

undefined and null are special values in JavaScript. They‚Äôre used to represent the absence of a value.

undefined is used when a variable has been declared, but not assigned a value. You shouldn‚Äôt deliberately set something to undefined.

null is used when a variable is deliberately left blank. It‚Äôs a way to say ‚Äúthis variable intentionally empty‚Äù. It‚Äôs useful for clearing any value that you might not want to keep anymore.

console.log(name); // Output: undefined

let age = null;
console.log(age); // Output: null

Comparisons

Comparisons make use of and return a boolean value. If you compare two numbers, you get a boolean in return: true or false.

Is 1 greater than 2? false.

Is 2 greater than 1? true.

Are they the same? false.

‚õ≥Ô∏è Play with the numbers and log the results to the console.

let a = 1;
let b = 2;
let isAGreaterThanB = a > b;
let isALessThanB = a < b;
let isAEqualToB = a === b;

console.log(isAGreaterThanB); // Output: false
console.log(isALessThanB); // Output: true
console.log(isAEqualToB); // Output: false

You can use string literals to log out more details about your variables:

console.log(`Is ${a} greater than ${b}?`, isAGreaterThanB); // Output: Is 1 greater than 2? false
console.log(`Is ${a} less than ${b}?`, isALessThanB); // Output: Is 1 less than 2? true
console.log(`Is ${a} equal to ${b}?`, isAEqualToB); // Output: Is 1 equal to 2? false
