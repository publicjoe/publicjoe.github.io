CSS Styling Changes (styles.css)

We are going to add the style for the new panel, plus put in a split so that the console can be resized.

First, let's put the editor-container back to ow it originally was

styles.css 
.editor-container {
    flex: 1;
    display: none;
}

Now add the following code after editor-container.active

styles.css 
/* Right panel: preview + console */
#right-panel {
  display: flex;
  flex-direction: column;
  height: 100%;
}

#preview-panel {
  flex: 2;
  position: relative;
}

Now let's make the background of the preview fram white

styles.css 
#preview-frame {
  width: 100%;
  height: 100%;
  border: none;
  background: white;
}

Now we can add the style for the console panel

styles.css 
#console-panel {
  flex: 1;
  background: #1e1e1e;
  color: #eee;
  font-family: monospace;
  font-size: 13px;
  padding: 8px;
  overflow-y: auto;
  border-top: 1px solid #444;
}

.console-log   { color: #0f0; }
.console-error { color: #f55; }
.console-warn  { color: #ff0; }
.console-clear { color: #888; font-style: italic; }

Finally, we need to add the style for the vertical resizer. So add the following code at the end of the file

styles.css 
.gutter.gutter-vertical {
  cursor: row-resize;
  height: 6px;
}
