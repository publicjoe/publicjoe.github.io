CodeFiddle

Requirements

So I thought about creating a JSFiddle / Codepen clone and how I would build it. Without getting into a deep design, I put down some simple initial requirements.

Overview

  • Left Panel → Tabs for each file (HTML, CSS, JS, etc.) and a code editor area for each tab.
  • Right Panel → An <iframe> that shows the live preview generated from combining the code in each tab.
  • Trigger → As the user edits code, or clicks “Run”, the right panel reloads the iframe with the combined code.

Tech Stack

  • HTML/CSS — Layout and structure.
  • JavaScript — Switching tabs, handling code updates, rendering preview.
  • CodeMirror or Monaco Editor — For a real code editor experience (syntax highlighting, indenting, etc.).

After reading about the two editors, Monaco seem the easiest to get started with.
