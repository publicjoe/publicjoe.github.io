CodeFiddle is a web-based code editor that allows you to write and preview HTML, CSS, JavaScript, and TypeScript code in real-time. This tutorial will guide you through building your own mini version of CodeFiddle using the Monaco Editor and Split.js for resizable panels.

By the end of this tutorial, you'll have a functional code editor with live preview capabilities, console logging, and more!
