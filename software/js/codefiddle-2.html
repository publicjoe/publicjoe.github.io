CSS Styling Changes (styles.css)

In order to allow use to resize the split-panel interface using the split library, we are going to make some changes to the style.

Remove the following code

/* Left: Editor */
#editor-panel {
    display: flex;
    flex-direction: column;
    width: 50%;
    border-right: 1px solid #ccc;
}

And replace it with this code that keeps code that is common to the two panels

styles.css 
/* Panel styles */
#editor-panel, #preview-panel {
    height: 100%;
    overflow: hidden;
}

This code sets the height of both panels to 100% of their container and hides any overflow content.

We do this because the Split.js library will handle the width of the panels for us.

overflow: hidden; ensures that any content that exceeds the panel's dimensions is not visible, which is useful for maintaining a clean layout when resizing.

 

Then replace the code in the editor-container with the following code that allows us to calculate the height of the editor.

styles.css 
.editor-container {
    height: calc(100% - 34px); /* 34px = tab height */
    display: none;
}

This code sets the height of the editor container to be the full height of its parent minus 34 pixels, which is the height of the tab bar above it.

This ensures that the editor fits perfectly within the available space below the tabs.

The display: none; is used to hide the editor container when it is not active.

 

Now remove the fixed width for the preview panel

/* Right: Preview */
#preview-panel {
    width: 50%;
}

We do this because the Split.js library will handle the width of the panels for us.

 

Now add this code that styles the resizer at the end of the file

styles.css 
/* Resizer style */
.gutter {
    background-color: #ddd;
    background-repeat: no-repeat;
    background-position: 50%;
}
.gutter.gutter-horizontal {
    cursor: col-resize;
    width: 6px;
}

This code styles the resizer handle that appears between the two panels when using Split.js.

The .gutter class sets a light gray background color for the resizer handle and centers any background image (if used).

The .gutter.gutter-horizontal class specifically styles horizontal gutters (the vertical resizer between left and right panels).

It changes the cursor to a horizontal resize cursor when hovering over the gutter, indicating that it can be dragged to resize the panels.

It also sets the width of the gutter to 6 pixels, making it easier to grab and drag.

What This CSS Does:

  • Remove: Any code that makes the containers fixed in width
  • Add: The height calculator to allow us to add a resizer
  • .gutter: The base style for all resizer handles
  • .gutter.gutter-horizontal: makes the draggable area 6 pixels wide and shows a bidirectional horizontal arrow cursor

JavaScript Functionality (script.js)

Now all we need to do is add the resizer into the javascript. Do this by adding the following code after let editors = {}; near the top of the file before any other code.

script.js 
// Set up resizable split between editor & preview
Split(['#editor-panel', '#preview-panel'], {
  sizes: [50, 50],
  minSize: 200,
  gutterSize: 6,
  cursor: 'col-resize'
});

This code initializes a resizable split layout between the editor panel and the preview panel using the Split.js library.

The Split function takes two arguments: an array of CSS selectors for the elements to be split, and an options object to configure the behavior of the split.

In this case, we are splitting the elements with IDs editor-panel and preview-panel.

The options object specifies the following settings:

  • sizes: [50, 50] - Sets the initial sizes of the two panels to be equal (50% each).
  • minSize: 200 - Sets the minimum size of each panel to 200 pixels, preventing them from being resized too small.
  • gutterSize: 6 - Sets the width of the gutter (the draggable area between the panels) to 6 pixels.
  • cursor: 'col-resize' - Changes the cursor to a horizontal resize cursor when hovering over the gutter, indicating that it can be dragged to resize the panels.

This allows users to dynamically adjust the width of the editor and preview panels by dragging the gutter between them.

What This JavaScript Does:

  • Initialise the resizer: Initializes Split.js on two elements with IDs editor-panel and preview-panel. Creates a draggable gutter between them for resizing
  • Configuration: Sets up the elements of what the resizer looks like
