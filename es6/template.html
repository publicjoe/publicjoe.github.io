Template Literals

In ES6, Template Literals are a new way to work with strings. In simple terms, you just use backticks instead of single or double quotes.

Example

const a_string = `something`

Some of the benefits of Template Literals are:

  • easier to create multiline strings
  • provides an easy way to interpolate variables and expressions in strings

Multiline strings

Once a template literal is opened with the backtick, you just press enter to create a new line, with no special characters.

Example

const string = `Here is a 
multiline string`;

Interpolation

Template literals provide an easy way to interpolate variables and expressions into strings. You do so by using the ${...} syntax:

Example

const myVariable = 'apples'
const string = `I like to eat ${myVariable}` // Output: I like to eat apples
