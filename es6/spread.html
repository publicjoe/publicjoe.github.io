Spread Operator

ES6 provides a new kind of parameter called the spread parameter that has a prefix of three dots (...). The spread operator allows you to spread out elements of an iterable object such as an array, a map, or a set.

Example

const numbers = [31,32,33];
const arr = [22,44,66, ...numbers];
console.log(arr); // Output: [ 22, 44, 66, 31, 32, 33 ]

So the three dots (...) represent both the spread operator and the rest parameter. We can think of the diffence between the two as:

  • The spread operator unpacks elements from an iterable object.
  • The rest operator packs elements into an array.

The rest parameter must be the last argument of a function. However, the spread operator can be anywhere:

const numbers = [31,32,33];
const arr = [...numbers, 22,44,66];
console.log(arr); // Output: [ 31, 32, 33, 22, 44, 66 ]

Array push

A good example to how the spread operator can be used is in the push() method.

Example without using spread

const numbers = [31,32,33];
const arr = [22,44,66];
arr.push(numbers);
console.log(arr); // Output: [ 22, 44, 66, [ 31, 32, 33 ] ]

The array is pushed into the array as an array.

Example using spread

const numbers = [31,32,33];
const arr = [22,44,66];
arr.push(...numbers);
console.log(arr); // Output: [ 22, 44, 66, 31, 32, 33 ]

The contents of the array are pushed into the array.
