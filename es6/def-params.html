Default Parameters

In JavaScript, a parameter has a default value of undefined. It means that if you don’t pass the arguments into the function, its parameters will have the default values of undefined.

Prior to ES6, you need to test a parameters value and assign a default value if it is undefined:

function say(message) {
    message = typeof message !== 'undefined' ? message : 'Hi';
    console.log(message);
}
say(); // 'Hi'

ES6 provides an easier way to set the default values for the parameters of a function as shown in the following syntax:

function say(message='Hi') {
    console.log(message);
}

say(); // Output: Hi

Passing in undefined

If you have a function with more than one default parameter, and you only want to overwrite one of the default parameters, then if it is not the first parameter that you wish to overwrite, then you can pass in undefined for the preceding paramters that you wish to leave alone.

Example

function add(x = 1, y = 2, z = 3) {
    return x + y + z;
}

console.log(add(undefined, 5)); // Output: 9

The arguments object

The value of the arguments object inside the function is the number of actual arguments that you pass in.

Example

function add(x, y = 1, z = 2) {
    console.log( arguments.length );
    return x + y + z;
}

add(10); // Output: 1
add(10, 20); // Output: 2
add(10, 20, 30); // Output: 3
