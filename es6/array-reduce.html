Array reduce

The JavaScript Array type provides the reduce() method that allows you to

Example

let employees = [
                {name: 'Dave', salary: 20000},
                {name: 'Dozey', salary: 19500},
                {name: 'Beaky', salary: 18000},
                {name: 'Mick', salary: 21500},
                {name: 'Titch', salary: 19000}
            ];
            
            const totalSalary = employees.reduce(function (acc, curr) {
                return acc + curr.salary;
            }, 0);
            
            console.log(totalSalary);

In ES6, this code looks like this in an arrow function combined with the map function.

let employees = [
                {name: 'Dave', salary: 20000},
                {name: 'Dozey', salary: 19500},
                {name: 'Beaky', salary: 18000},
                {name: 'Mick', salary: 21500},
                {name: 'Titch', salary: 19000}
            ];
            
            const empSalaries = employees.map((emp) => emp.salary);
            
            const totalSalary = empSalaries.reduce((sum,value)=>sum+value, 0);
            
            console.log(totalSalary);
