Rest Operator

ES6 provides a new kind of parameter called the rest parameter that has a prefix of three dots (...). The rest parameter allows you to represent an indefinite number of arguments as an array.

Example

function foo(a,b,...args) {
    //...
}

A rest parameter can only be the last parameter of a function. It behaves like an array, containing all of the properties of an array object.

In the previous example, the first argument maps to a, the second one maps to b, and the third, the fourth, etc., will be stored in the rest parameter args as an array. so if we called the function with the following arguments:

foo(1,2,3,'A',4,'B',5,'C');

Then the args array stores the following values:

[3,'A',4,'B',5,'C']

So if you wanted to add up only the numbers in the args array we could write a sum function that takes advantage of the rest operator.

Example

function foo(a,b,...args) {
    console.log(sum(...args));
}

function sum(...args) {
    return args.filter(e => typeof e === 'number')
        .reduce((prev, curr)=> prev + curr);
}

foo(1,2,3,'A',4,'B',5,'C'); // Output: 12
