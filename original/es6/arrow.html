Arrow Functions

ES6 arrow functions provide you with an alternative way to write a shorter syntax compared to the standard function expression.

Example

Here is a function expression:

let add = function(x,y) {
    return x + y;
}
console.log(add(10,20)); // Output: 30

Now here is the equivalent to the above using an arrow function:

let add = (x,y) => x + y;
console.log(add(10,20)); // Output: 30

If more code is needed in the body of the function and you need to use block syntax, then you need to specify the return keyword:

let add = (x,y) => { return x + y };
console.log(add(10,20)); // Output: 30

Syntax

If an arrow function has two or more parameters, you use the following syntax:

(p1, p2, ..., pn) => { statements };

If an arrow function takes a single parameter, you use the following syntax:

p => { statements }

If the arrow function has no parameters, then you must use the parentheses:

() => { statements }

The parameter definition and the arrow (=>) in an arrow function must be on the same line.

If you use a simple expression in the body of an arrow function, you don’t need to use the curly braces.

let square = x => x * x;

However, if you use a statement, you must wrap it inside a pair of curly braces as in the following example:

let except = msg => {
    throw msg;
};

Object Literals

Since both block and object literal use curly brackets, the JavasScript engine cannot distinguish between a block and an object. To fix this, you need to wrap the object literal in parentheses as follows:

let setColor = color => ({ value: color });

Differences between functions and arrow functions

  • An arrow function doesn’t have the arguments object.
  • An arrow function doesn’t have the prototype property.

JavaScript rest parameters and arrow function

Because the arrow function does not have the argumentss object, if you want to pass a number of arguments to the arrow function, you must use the rest parameters. See the following example:

const combine = (...args) => {
    return args.reduce((prev, curr) => prev + ' ' + curr);
};
let message = combine('JavaScript', 'Rest', 'Parameters'); // =>
console.log(message); // Output: JavaScript Rest Parameters
